브라우저 자동화

잡 검색·기업 조사·공고 본문 추출에 사용

잡 검색·회사 조사·본문 추출 크롤링에 사용. 로그인된 상태로 두면 조사 시 해당 세션을 활용합니다.

설정을 바꾼 뒤 저장하면 자동 재시작됩니다. 또는 즉시 재시작:
사이트 로그인용 Chrome 열기 — 크롤용 Chrome을 일반 창으로 띄움. 구글·사람인·잡코리아 등에 한 번 로그인하면 쿠키가 저장돼서 이후 크롤에 자동 활용됩니다 (reCAPTCHA 회피).
쿠키·세션·캐시 전부 삭제 — 크롤 Chrome 프로필 디렉터리를 통째로 비웁니다. 로그인 정보 모두 사라짐. 차단당했을 때나 초기화하고 싶을 때 사용.

평소 쓰는 Chrome을 디버그 포트로 실행해두면, 우리 앱이 그 Chrome 세션을 그대로 빌려씁니다. 이미 로그인된 사이트(GitHub, LinkedIn 등)도 그대로 접근 가능. 아래 URL을 채우면 위 자동 모드보다 우선 적용됩니다.

실행 명령 (Windows PowerShell)
& "C:\Program Files\Google\Chrome\Application\chrome.exe" --remote-debugging-port=9222
macOS
/Applications/Google\ Chrome.app/Contents/MacOS/Google\ Chrome --remote-debugging-port=9222

⚠ 평소 쓰던 프로필을 그대로 쓰려면 모든 Chrome 종료 후 실행. 프로필 lock 충돌 주의.
